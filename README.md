# Website Booking Futsal
Tugas kuliah sistem booking sederhana menggunakan HTML, CSS, JS, PHP, dan MySQL.

## Cara Instalasi
1. Import file `db_futsal.sql` di phpMyAdmin.
2. Letakkan folder di dalam `htdocs`.
3. Akses via `localhost/futsal-booking`.

---

## 🔗 Link Akses (Lokal)
* **Halaman User:** `http://localhost/futsal-booking/`
* **Halaman Admin:** `http://localhost/futsal-booking/admin.php`
